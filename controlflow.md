# Control Flow
Because we need to!

## IF

```
a=3
if a<3 :
    print("A is less than 3")
elif a==3 :
    print("A is equal to 3")
else:
    print("A is greater than 3")
```

## Loops
```
while expression :
    do someting
else :
    do something else
```

```
for var in list :
    do something
else :
    do something else
```

## Error Handling
```
try:
    do something
except
```

## Context blocks
```
with expression as variable, expression as variable :
    do something with the variables
```
